# 🚀 Auto File Organizer v1.0.1 (Update Patch)

Aplikasi pintar pengelola file otomatis kini hadir dengan versi yang lebih sempurna! Berkat masukan Anda, kami telah meningkatkan performa dan fungsionalitas UI.

## 🌟 What's New in v1.0.1
- **📑 Navigasi Top Bar Baru:** Tab Dashboard, Logs, Tutorial, dan About kini dipindahkan ke Top Bar (Atas). Ini membuat desain jauh lebih bersih dan tidak lagi menghalangi tombol Quit di bagian bawah!
- **🔄 Import & Export Settings:** Sekarang Anda dapat menyimpan pengaturan (Rules, Bahasa, Tema, dll) ke dalam file `.json` dan memuatnya kembali di komputer lain dengan satu klik! Tombol berada di Top Bar bagian kanan.
- **📚 Tutorial & About yang Dinamis:** Teks Tutorial dan About kini mendukung multi-bahasa secara instan, dan link GitHub kini dapat langsung diklik untuk membuka browser!
- **🛠️ Bug Fix Kritis (TclError):** Memperbaiki masalah crash yang sering terjadi (`_tkinter.TclError: invalid command name`) saat mengubah tema atau memperbarui aturan sambil membuka Dashboard. Transisi UI sekarang dijamin 100% mulus dan aman.

## 📦 Downloads
Pilih versi yang paling sesuai dengan Anda:
1. **Portable Version (`AutoFileOrganizer_Portable.exe`)**
   - Aplikasi siap pakai dalam satu file *standalone*. Sangat cocok untuk dibawa di Flashdisk!
2. **Full Installer (`AutoFileOrganizer_Installer.exe`)**
   - Cara paling rapi untuk menginstall di komputer Anda. Membuat shortcut di Desktop dan bisa di-uninstall lewat Control Panel.

> **Catatan:** Terkadang Antivirus memblokir file `.exe` yang baru dibuat. Anda bisa dengan aman menekan "More Info" -> "Run Anyway".
